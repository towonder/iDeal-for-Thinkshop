SETUP

Drag & drop in the thinkshop plugins-folder. Edit /ideal/vendors/security/config.conf and add
the certificates needed for ideal in the same folder. Then activate the plugin using Thinkshop's
pluginpanel in the admin

You can find out how to generate certificates over here:
http://www.ideal-simulator.nl/ideal-professional-ssl-certificaten.html
